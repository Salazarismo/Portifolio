## 🚀 Featured Projects

### Habit Analyzer
More than just a digital checklist, Habit Analyzer is a web application designed to track and decode your daily routines with detailed analytics and reports. Built with a FastAPI backend and a Next.js/TypeScript frontend, it tackles the problem of scattered manual tracking by centralizing data and providing per-habit history. By implementing pre-aggregation and caching mechanisms, I managed to reduce the analytics read cost by 40% for large time windows, ensuring a snappy dashboard experience.

### Nexus Platform
Nexus is a robust ticketing platform tailored for high-friction event environments. It features a complete WhatsApp sales flow, Mercado Pago payment integration, and a multi-tenant NestJS backend. Its biggest triumph is solving the "network drop at the gate" problem: I implemented an offline-first check-in PWA using IndexedDB for a local queue and batch synchronization. This guarantees 100% operational capability at the gate even without internet access, severely reducing operational risk on event day.

### Relatorium — Quick Activity Button
Developed specifically for rural management, this cross-platform Flutter module streamlines frequent data entry tasks in the field (like activities, soil analysis, and budgeting). To solve the issue of buried menus and slow navigation, I encapsulated standardized shortcuts into a reusable `QuickActionButton` using Semantics for accessibility, Tooltips for feedback, and `go_router` for predictable named routes. This architectural UI improvement reduced the time to log field activities by 45–60% (from 45 seconds down to 20–25 seconds).
